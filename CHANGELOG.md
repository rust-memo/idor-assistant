# Changelog

## 3.0.0

- Replaced fixed Profile A/B slots with multi-profile identity and role management.
- Added session-bound CSRF/XSRF substitutions and v2 profile migration.
- Added a bounded candidate catalog with up to 20 observations per endpoint/reference.
- Added observed-owner inference and masked object fingerprints.
- Added bounded XML, cookie, multipart, and expanded GraphQL discovery.
- Added content-aware response evidence and owner-baseline stability checks.
- Added sequential, scope-bound read-only Batch with request budgets and stop policies.
- Added manual, redacted Site Map issue creation and optional redacted report evidence.
- Split the UI into Dashboard, Profiles, Test Matrix, Evidence, and Settings.
- Replaced source-text safety tests with behavioral tests and added JaCoCo verification.

## 2.0.0

- Initial public evidence-driven two-profile workflow.
